纯shell写的贪吃蛇

不到两百行的贪吃蛇

纯bash写的，有点蛋疼就是bash仅支持一维数组，所以每个蛇身像素的xy坐标都是模拟的二维数组表示。

还有一个问题就是bash的变量在函数里面声明，在函数外面也可以访问和修改，工具无好坏之分，主要看怎样去利用。

每次调用new_food仅仅只是刷新里面food_xy的值，然后什么时候要用到就直接读取出来，其实更规范的写法应该是new_food外面定义个food_xy,然后调用new_food时候把值返回给food_xy。
还是怎么方便怎么写吧。

过于简陋，轻点喷。
